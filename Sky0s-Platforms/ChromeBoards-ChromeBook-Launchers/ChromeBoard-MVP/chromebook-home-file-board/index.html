Home File Board

This is the real ChromeOS workaround: a fake home screen app for your files. Install it as a PWA, add your local files, folders, or Drive/web links, and launch them from one clean board instead of fighting ChromeOS like it owes you rent.

ChromeOS still does not allow apps or shortcuts directly on the desktop wallpaper/background. This app sidesteps that by becoming the thing you put on the launcher/shelf and use like a file home screen.

Ready. Add files or folders, then install this as an app on your Chromebook.
