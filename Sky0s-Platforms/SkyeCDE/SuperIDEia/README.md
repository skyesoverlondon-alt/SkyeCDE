# SuperIDEia

SuperIDEia is the SkyeCDE-hosted upgrade lane for SuperIDE.

The original SuperIDE platform remains intact.
This lane now uses lane-owned modules for workspace defaults, file hints, runtime recipes, session identity, launch mapping, and layout.

Lane-owned source files
- src/superideia-workspace-bridge.js
- src/superideia-file-actions.js
- src/superideia-terminal-actions.js
- src/superideia-session-bridge.js
- src/superideia-launch-map.js
- src/superideia-layout.css
