# SkaixuProIDEia

SkaixuProIDEia is the SkyeCDE-hosted upgrade lane for SkaixuPro-IDE.

The original SkaixuPro-IDE platform remains intact.
This lane now uses lane-owned modules for workspace defaults, file hints, runtime recipes, session identity, launch mapping, and layout.

- src/skaixuproideia-workspace-bridge.js
- src/skaixuproideia-file-actions.js

- src/skaixuproideia-launch-map.js
- src/skaixuproideia-layout.css
- src/skaixuproideia-shell.js
