# SkyDexia Release Command Center

This file is the lane-owned checklist for a real SkyDexia release cycle.

## Required source surfaces
- current SkyDex
- SkyDexia shell
- SkyDexia launch map
- SkyDexia runtime recipes
- deployment systems guide
- runtime runbook

## Required artifacts
- release brief
- smoke report
- rollout plan
- environment audit
- incident log if needed
- release handoff

## Required proofs
- preserved SkyDex launch proof
- SkyDexia runtime start proof
- runtime log proof
- deployment contract review proof

## Release desk rule
Do not treat SkyDexia as complete because it renders. Treat it as complete only when it can prove its own runtime, evidence, and deployment flows from inside the lane.
