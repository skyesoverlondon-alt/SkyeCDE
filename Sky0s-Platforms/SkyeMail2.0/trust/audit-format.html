Audit Log Export Format

Two recommended formats: JSON Lines (JSONL) and CSV.

JSONL record

{"ts":"2026-02-22T17:12:33.120Z","org_id":"...","actor_user_id":"...","actor_email":"...","event":"MESSAGE_READ","target_type":"message","target_id":"...","ip_hash":"...","meta":{"key_version":3,"thread_id":"..."},"retain_until":"2027-02-22T00:00:00.000Z"}

CSV columns

ts,org_id,actor_user_id,actor_email,event,target_type,target_id,ip_hash,retain_until,meta_json

The meta_json field is a serialized JSON object for extensibility.
