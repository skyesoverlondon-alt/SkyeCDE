Subprocessors

Subprocessors are optional depending on configuration. Enable only what you use.

Netlify
Hosting, CDN, serverless functions, scheduled functions.
Neon Postgres
Primary database for ciphertext storage, org/RBAC, audit logs, rate events.
Resend
Email delivery for verification, reset, and notifications.
Cloudflare Turnstile
Bot defense for public send + thread reply (optional).
AWS KMS
Encrypts managed keys and protected config secrets (optional).
Postmark
Inbound email webhook import (optional).
Splunk / Datadog
SIEM export targets (optional). Tokens are encrypted at rest.
Kaixu Gateway
AI calls route through your Kaixu Gateway only (optional).

Customers may request a signed subprocessor addendum and notice periods via contract.
